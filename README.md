# Solfege Labels Plugin for MuseScore 3

A MuseScore 3 plugin that labels selected notes with solfege syllables (do, re, mi, fa, sol, la, ti) using movable do relative to the current key signature.

## Installation

1. Download `doremi.qml`
2. Copy it to your MuseScore plugins folder:
   - **macOS**: `~/Documents/MuseScore3/Plugins/`
   - **Windows**: `%HOMEPATH%\Documents\MuseScore3\Plugins\`
   - **Linux**: `~/Documents/MuseScore3/Plugins/`
3. In MuseScore, go to **Plugins > Plugin Manager**
4. Enable "Solfege Labels"

## Usage

1. Select one or more measures in your score
2. Go to **Plugins > Solfege Labels**
3. Staff text labels will be added below each note
